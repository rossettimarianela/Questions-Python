## Instalar Dependencias - Python3 y Jupyter Notebook
- Abrimos la consola con (ctrl + alt + T) o buscando console o cmd en la barra de busquedas
### Python3
```bash
sudo apt update
sudo apt install python3
```

### Jupyter Notebook
```bash
pip install notebook
```
Si pip no está instalado:
```bash
sudo apt install python3-pip
```

## Ejecución
1. Clonar el repositorio:
```bash
git clone 
```
2. Navegar hasta la carpeta del proyecto:
```bash
cd Practica_2/Entrega_2
```
3. Abrir Jupyter Notebook:
```bash
jupyter notebook
```
4. En el navegador navegar hasta la carpeta donde se encuentra descargado `Practica_2/ Entrega_2` y abrir `P2_Ejercicio10.ipynb`
5. Ejecutar todas las celdas con **Run All** o celda por celda con (ctrl + Enter) sobre la misma.
